# Synapse

Mapa mental interativo 3D para visualizar e conectar ideias.

## Como abrir

### Opção 1 — Servidor local (recomendado)

```bash
cd synapse && python3 -m http.server 8080
```

Depois abra http://localhost:8080 no navegador.

### Opção 2 — GitHub Pages

https://httpweez.github.io/synapse/

(Verifique se o GitHub Pages está ativado em Settings > Pages do repositório.)

## Atalhos

| Tecla | Ação |
|-------|------|
| `N` | Nova ideia |
| `R` | Reorganizar grafo |
| `F` | Modo foco |
| `Del` | Excluir nó selecionado |
| `Esc` | Fechar painel/modal |
